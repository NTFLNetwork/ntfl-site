# NTFL Network

A GitHub Pages-ready NTFL site starter built from your uploaded schedule workbook.

## What is included
- Home page
- Team pages with full schedules
- Week-by-week schedule page
- Auto standings
- Hall of Fame
- History
- Single-admin dashboard with local editing, export, and import

## Files
- `index.html`
- `teams.html`
- `team.html`
- `schedule.html`
- `standings.html`
- `hof.html`
- `history.html`
- `admin.html`
- `assets/data.js`
- `assets/site.css`
- `assets/site.js`

## How to use
1. Upload the folder to GitHub.
2. Turn on GitHub Pages.
3. Open `admin.html` to edit the homepage, schedule scores, primetime tags, HOF, history, and team bios.
4. Use **Export JSON** to back up the current browser state.

## Important note
This version stores admin edits in the browser on the device that made them. That is perfect for a starter and for building the site now, but if you want edits to sync across devices and sessions, the next step is connecting Supabase or another backend.

## Schedule data
The schedule was converted from your uploaded workbook into the built-in JSON data file.
