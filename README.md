# NTFL Network Starter

This starter is built for GitHub Pages and uses a single-admin, browser-saved workflow.

## Included
- Home
- Teams
- Team detail pages
- Schedule
- Standings
- Rankings
- Hall of Fame
- History
- Admin page with local editing

## Logo
This build is wired to use `IMG_5900.png` in the repository root.

## What to do
1. Upload all files to GitHub.
2. Put your logo image in the repo root as `IMG_5900.png`.
3. Turn on GitHub Pages.
4. Use `admin.html` to edit site text and rankings.
